# IOT Health Platform

# Dependencies
- `docker` and `docker-compose`

# Start server 
Run `docker-compose -f ./docker-compose-dev.yml up -d` on the main folder

Clear cache with : `docker-compose -f ./docker-compose-dev.yml up -d --build --force-recreate`
